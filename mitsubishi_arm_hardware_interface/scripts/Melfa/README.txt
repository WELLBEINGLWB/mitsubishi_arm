1.prg is the Melfa Basic V program that needs to run on the controller so that the servo is turned on and external communication is enabled. This program has to be started from the controller panel before the ROS node that communicates with the robot.

The PC connected to the controller via ethernet has to have an IP address of 192.168.0.3.

CAUTION:
Always have the emergency stop button available nearby in case something unexpected happens.
